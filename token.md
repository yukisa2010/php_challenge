# 1. 初期表示
  - index.phpなど、初期表示画面にtokenを埋め込む
  - INSERTする
    - sha1などで加工したtoken
    - 有効期限
# 2. ログインボタンクリック時
  - tokenをsubmitでPHPに渡す。
  - DBからSELECTでtokenの存在を確認する(有効期限も確認/sha1で再度加工してから探す)
  - 認可OKの場合
    1. id, password(sha1で加工)が一致するか確認
    1. 一致していればセッションスタート
# 3. その後のセッションキープについて
  - アクセスごとにデータベースをいじるパターン
    - 有効期限は都度書き換える。5分をベースに作ってみる
